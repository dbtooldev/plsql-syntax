# Support

If you need help with PL/SQL Syntax Highlighting, use one of these channels:

- Email: dbtool@outlook.com
- GitHub issues: https://github.com/dbtooldev/plsql-syntax/issues

When reporting a problem, include:

- Your editor and version
- The extension version
- A short PL/SQL or SQL sample that reproduces the issue
- A screenshot if the highlighting behavior is unclear
